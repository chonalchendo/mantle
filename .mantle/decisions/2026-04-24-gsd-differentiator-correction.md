## Decision

Revised the GSD differentiator row in product-design.md. Removed the false claim 'GSD has no runtime' and led with Mantle's real edges: persistent cross-project Obsidian-native knowledge graph, and the cohesive idea-to-review lifecycle.

## Alternatives Considered

- Keep the existing 'no runtime' framing (rejected: factually false)
- File as a backlog issue and run /mantle:revise-product later (rejected: corrections aren't work, this is a one-edit copy fix)

## Rationale

The 2026-04-22 GSD scout (.mantle/scouts/2026-04-22-get-shit-done.md) confirmed GSD's sdk/src/ contains a full Anthropic SDK runtime: session-runner, context-engine, cli-transport, event-stream, and tool-scoping. The 'no runtime' positioning was demonstrably wrong and undermined Mantle's credibility against a real competitor. Mantle's actual differentiators — persistent cross-project knowledge graph + Obsidian-native + cohesive lifecycle — survive direct comparison and are more honest.

## Reversal Trigger

If GSD adds a wikilinked cross-project knowledge graph that integrates with Obsidian, or if Mantle drops the persistent vault for a per-repo planning model, the differentiator framing needs to change again.
